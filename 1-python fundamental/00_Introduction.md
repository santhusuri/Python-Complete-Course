## What is Python?

Python is a high-level, interpreted programming language designed for readability and simplicity. It emphasizes code readability with its clean syntax, making it an excellent choice for beginners and experienced developers alike. Python supports multiple programming paradigms, including procedural, object-oriented, and functional programming.

### Key Features:
- **Easy to Learn**: Simple syntax similar to English.
- **Versatile**: Used for web development, data science, automation, AI, etc.
- **Large Standard Library**: Extensive built-in modules for various tasks.
- **Cross-Platform**: Runs on Windows, macOS, Linux.
- **Community Support**: Huge community and ecosystem.

### History:
- Created by Guido van Rossum in 1989 at the Centrum Wiskunde & Informatica (CWI) in the Netherlands.
- First released in 1991.
- Named after the British comedy group Monty Python.
- Python 2.0 released in 2000, Python 3.0 in 2008 (major overhaul for better consistency).

## Python Versions & Installation

### Versions:
- **Python 2.x**: Legacy, no longer supported (end of life in 2020).
- **Python 3.x**: Current stable version (e.g., 3.8, 3.9, 3.10, 3.11). Always use the latest stable version.

### Installation:
1. Download from [python.org](https://www.python.org/downloads/).
2. Follow the installer instructions.
3. Verify installation: Open terminal and run `python --version`.
4. Optional: Use package managers like `pip` for libraries.

## Running Python Scripts

### Command Line Interface (CLI):
- Save code in a `.py` file (e.g., `hello.py`).
- Run: `python hello.py`

### IDLE (Integrated Development and Learning Environment):
- Comes with Python installation.
- Interactive shell for quick testing.
- File editor for scripts.

### VS Code:
- Install Python extension.
- Open folder, create `.py` files, run with F5 or terminal.

### Jupyter Notebooks:
- Interactive environment for data science.
- Install via `pip install jupyter`.
- Run: `jupyter notebook`
- Cells for code, markdown, and output.

### Interactive Shell:
- Run `python` in terminal for REPL (Read-Eval-Print Loop).
- Type code directly and see results instantly.
